参数

1. alertText

  • 名称：alert text
  • 描述：要检查的文字

2. answer

  • 名称：answer
  • 描述：给出响应弹出提示的答案。

3. attributeLocator

  • 名称：attribute locator
  • 描述：一个元素定位器，后跟一个@符号，然后是属性的名称，例如“foo @ bar”。

4. arrayVariableName

  • 名称：array variable name
  • 描述：包含 JavaScript 数组的变量的名称。

5. conditionalExpression

  • 名称：conditional expression
  • 描述：JavaScript 表达式，返回用于控制流命令的布尔结果。

6. coord

  • 名称：coord string
  • 描述：指定鼠标事件相对于从定位器找到的元素的 x，y 位置（例如， - 10,20）。

7. expectedValue

  • 名称：expected value
  • 描述：您期望变量包含的结果（例如，true，false或其他值）。

8. expression

  • 名称：expression
  • 描述：您要存储的值。

9. formLocator

  • 名称：form locator
  • 描述：要提交的表单的元素定位器。

10. handle

  • 名称：window handle
  • 描述：表示特定页面（选项卡或窗口）的句柄。

11. iteratorVariableName

  • 名称：iterator variable name
  • 描述：在循环控制流命令中迭代集合时使用的变量的名称（例如，对于每个）。

12. json

  • 名称：json
  • 描述：JavaScript 对象的字符串表示形式。

13. keySequence

  • 名称：key sequence
  • 描述：要键入的键序列，可用于发送键击（例如$ {KEY_ENTER}）。

14. locator

  • 名称：locator
  • 描述：元素定位器。

15. locatorOfDragDestinationObject

  • 名称：locator of drag destination object
  • 描述：元素的定位符，其位置（例如，其中最中心的像素）将是要拖动的对象的定位符被丢弃的点。

16. locatorOfObjectToBeDragged

  • 名称：locator of object to be dragged
  • 描述：要拖动的元素的定位器。

17. loopLimit

  • 名称：loop limit
  • 描述：可选参数，指定循环控制流命令可以执行的最大次数。这可以防止无限循环。默认值设置为1000。

18. message

  • 名称：message
  • 描述：要打印的消息。

19. optionLocator

  • 名称：option
  • 描述：选项定位器，通常只是一个选项标签（例如“John Smith”）。

20. pattern

  • 名称：text
  • 描述：精确的字符串匹配。对模式匹配的支持正在进行中。有关详细信息，请参阅 https://github.com/SeleniumHQ/selenium-ide/issues/141。

21. region

  • 名称：region
  • 描述：指定一个带坐标和长度的矩形（例如，“x：257，y：300，width：462，height：280”）。

22. resolution

  • 名称：resolution
  • 描述：使用 WidthxHeight 指定窗口分辨率。（例如，1280 x 800）。

23. script

  • 名称：script
  • 描述：要运行的 JavaScript 代码段。

24. selectLocator

  • 名称：select locator
  • 描述：标识下拉菜单的元素定位器。

25. testCase

  • 名称：test case
  • 描述：项目中的测试用例名称。

26. text

  • 名称：text
  • 描述：要验证的文本。

27. times

  • 名称：times
  • 描述：一次控制流循环的次数将执行其块内的命令。

28. url

  • 名称：url
  • 描述：要打开的 URL（可以是相对的或绝对的）。

29. value

  • 名称：value
  • 描述：要键入的值。

30. variableName

  • 名称：variable name
  • 描述：没有括号的变量的名称。

31. waitTime

  • 名称：wait time
  • 描述：等待的时间量（以毫秒为单位）。

32. xpath

  • 名称：xpath
  • 描述：要评估的 xpath 表达式。
